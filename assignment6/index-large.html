Magnifier Challenge

When I was using the magnifier I had two problems. The first one is that when I was using the full screen mode, there is no way for me to move my zoom focus point. Whenever I want ot move my focus I have to zoom out and move my cursor then zoom in again. I think this could be improved. Another problem is that when I see the website in scaled mode it was hard to find some content in the web. Since everything was scaled it took a lot of time to navigate. I think in this mode an modification could be made is that we can reduce irrelavent item in the website and only scale text part.

About This Project

To make my web a low Vision mode, first I increased all of my text size. Then I set the background color to black and set my font color to white. I also decreased the margin size of all of my button to make them closer to each other. I changed my hover function color from green to white and at the same time the color of text was change to black when hover function is initiated.
